## 五、关键发现

1. **SJTU摄影占比畸高（61.6%）**：vs 清华20.3%、北大4.2%、MIT 30.0%。大量案例本质是新闻配图而非科学可视化，存在数据转化率低的问题。

2. **数据可视化严重缺失（4.6%）**：清华（42.3%）和北大（39.2%）以此为主力，SJTU在此维度与国内同行差距悬殊，可能反映出科研宣传对定量信息呈现方法论重视不足。

3. **functionalPurpose标注率仅24.9%**（76/305）：除医学（91.7%标注）外，其余学科大量空值。该字段的编目标注是后续分析的瓶颈。

4. **医学是内部最均衡的学科**：显微图37.5% + 摄影37.5% + 数据可视化20.8%，三种呈现方式均有覆盖，且functionalPurpose涵盖数据/记录/传播/解释四类。

5. **自动化采集偏向性明显**：案例主要来自news.sjtu.edu.cn（97条，31.8%），新闻来源天然偏爱摄影类配图，可能导致数据采集系统性偏向。

## 六、数据不足与改进建议

| 问题 | 建议 |
|------|------|
| 化学(5条)、环境科学(2条)样本严重不足 | 需专项采集化学化工学院、环境学院网站的科研展示页面 |
| 物理、生命科学无显微图（物理应有电镜、生命科学应有共聚焦等） | 定向补充实验室/课题组页面的原始科研图片 |
| 3D渲染、混合媒介零案例 | 建议纳入科研动画截图、分子渲染图、VR/AR展示等新形式 |
| functionalPurpose标注率低 | 需安排专项标注，先补SJTU案例的functionalPurpose字段 |
| 缺少distributionMedium维度分析 | 数据中超过70%该字段为空，暂无法做静图/动图/交互式分层 |
| 对标数据非全量且来源不同 | 清华/北大/MIT数据以各自校内新闻站为主，来源结构差异未控制，对比仅供参考 |
