## D-UI. 三栏对比视图系统功能

### 目标

在 Sci-Viz Case Hub 的数据分析页面中，实现"交大学科 × 三栏对比"的可视化功能，使用户能直观看到同一学科在交大、国内、国际三方的呈现方式差异。

### 核心交互

1. 用户在数据分析页面选择一个学科（如"工程"）
2. 页面展示三栏并排对比：
   - 左栏：交大现状（该学科在SJTU来源的分布）
   - 中栏：国内研究（该学科在清华/北大/复旦/中科大/中科院来源的分布）
   - 右栏：国际研究（该学科在Nature/MIT/Harvard等国际来源的分布）
3. 可展开第四栏：头部企业（待采集，当前为空数据占位）
4. 每栏展示选定维度（mediaType/functionalPurpose/distributionMedium等）的分布条形图
5. 条形可点击跳转到案例列表（带筛选条件）

### 具体验收标准

1. **后端API**：`GET /api/insights/comparison?discipline=工程&dimension=mediaType` 返回三组分布数据
2. **前端组件**：三栏并排布局，每栏包含标题、案例数、分布条形图
3. **学科选择器**：下拉选择学科，默认选中"工程"
4. **维度选择器**：下拉选择分析维度，默认"呈现方式"
5. **颜色区分**：三栏使用不同主题色（交大=紫、国内=绿、国际=橙）
6. **企业栏折叠/展开**：默认折叠，点击展开显示空数据提示"待采集"
7. **点击跳转**：条形点击跳转到案例列表带筛选参数
8. **空状态和加载态**：无数据显示提示，加载中显示loading
9. **编译通过**：server + web 均编译无错
