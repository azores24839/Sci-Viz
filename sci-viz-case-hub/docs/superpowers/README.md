# Agent 文档说明

`docs/superpowers/` 存放面向 AI / agent 的规格、阶段目标和执行计划。这里的文档不是普通报告，通常带有任务边界、验收标准或交接说明。

## 目录分工

- `specs/`：阶段目标、设计说明、任务书、分类字典和专题资料。
- `plans/`：可执行计划，通常面向后续 agent 逐步实现。

## 使用规则

- 新建 agent 任务前，先检查 `../README.md` 中是否已有同类文档。
- 阶段目标使用 `YYYY-MM-DD-phase-*-goals.md` 命名。
- 执行计划使用 `YYYY-MM-DD-<topic>.md` 命名。
- 一次性执行报告优先放在 `../` 的报告区；只有与 agent 交接强相关时才放入 `specs/`。
- 如果文档已经转化为稳定方法论，后续可以汇总到核心文档，再保留原文件作为历史记录。
